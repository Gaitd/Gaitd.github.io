开言

这个算是我记录搭建博客的介绍吧

开始准备

  1. nodeJs

  2. github

  3. hexo

  4. git

所需软件大概就只有这些了。

一、nodejs

链接：https://nodejs.org/en/

官网看自己的电脑决定下那个操作系统，然后下载就一路默认就好了

下好后在系统环境变量给他一个路径。

此电脑右击属性点击高级系统设置在点环境变量点击Path；

一般情况应该是有路径，如在D盘就，D://文件名就ok了。

然后在命令行cmd中输入node -v和npm -v；

像这样就可以了

二、github

首先就要有一个GitHub的账号，注册就不写了。

有账号之后，点击头像旁边的加号，点击New repository，来新建一个仓库

然后进来这个界面

1的仓库名为你的用户名.github.io

2的选择是决定这个仓库是公共的还是私有的，私有的需要付费.

3的那个README file 是让你的这个仓库初始化，也可以理解为文档

拉到最后，点击绿色按钮提交，到这个界面点击settings

进去后拉到GitHub Pages这里，刚进去是没有这个绿色区域的链接的

点击Change theme选择一个主题，提交后，在进去就有这个链接了，点开是没有内容的，因为没有文件也没有输入。

三、git

链接：https://git-scm.com/

这个也没什么好说的就一路默认就好。

只注意以下，在某一步的选择路径的时候记得选择以下：Use Git from the Windows Command Prompt

这样就直接添加了路径可以在cmd里面打开它。

安装好后，验证以下:git –version

四、hexo

hexo就是重头戏了，搭建博客主要就是用这个了。

开始就是老规矩先下载了，不过这个就不需要链接了。

先在一个适合的地方（除了C盘，其他盘看着选一个），新建一个文件夹用来存放自己的博客文件，如：D:\blog

就可以了。然后在这个新建的文件里面右击选择Git Bash Here,来打开git的控制台窗口；

进去后输入npm install hexo-cli -g安装hexo,如果有报错先不急，输入 hexo -v 验证是否安装成功，成功就不用管了。

然后就是初始化我们的网站,输入 hexo init 初始化文件夹，接着输入npm install安装一些必备的组件。注意npm 是nodejs的命令，如果输入npm报错没有找到此命令,先看看nodejs有没有安装好，或者路径有没有配置。

没有错误，那本地网站配置就基本能好了，输入hexo g生成静态网页,输入hexo s 打开本地服务器，浏览器打开地址localhost:4000/,就能看到我们的博客了。

按住Ctrl + c 关闭我们的本地服务器

五、连接Github和hexo

前面创建github的仓库，按照我的理解就是hexo是一个模板，给你创建好博客的东西，让你更方便的管理和使用这个博客，而github的仓库就是可以把实时的把你的文件保存和维护，而且我们使用的是github仓库的这样一个域名，是不需要花钱的。

话不多说，开始。

首先右键打开git bash（应该是随便那个地方打开都行，不过最好还是在blog你的博客目录下打开吧），然后输入以下命令:

git config –global user.name “github的用户名”

git config –global user.email “注册邮箱”

然后生成密钥SSH key:

ssh-keygen -t rsa -C “注册邮箱”

打开github，在头像下面点击settings，进入后，在左边选中 SSH and GPG keys,新建一个SSH,名字随便了。

然后在git bash中输入

cat ~/.ssh/id_rsa.pub

除了输入你这个key的密码,其他直接回车，最后输出一大段的内容，把这段内容复制到新建SSH的那个框里面，点击确定保存，就把这个SSH创建好了。

然后在git bash中输入

ssh -T git@github.com

输入后需要输入你的key密码，成功后像这样就OK了。

到这一步，就很快了。

打开博客根目录，就是你打开git bash的那个目录，打开_config.yml文件，这是博客的配置文件，在这里你可以修改与博客相关的各种信息。

直接拉到最后，修改配置：

有的就不用的，按照这个更改就好，你的仓库地址要填对，不，这个是你的那个网页地址，就是github给你生成的那个链接输入进去，这样，上传到github才不会找不到地址.

六、最后的最后，写文章喽

​ 首先在博客根目录打开git bash ，安装一个扩展 npm i hexo-deployer-git。

然后输入hexo new post “你想输入的博客文章名”,新建一篇文章

回到根目录，打开\source\_posts的目录，下面多了一个.md文件，就是你新创建的文章,创建的文章基本都是在这个__posts的目录下。

你可以点进去编写一下,或者直接在根目录打开git bash，输入hexo g生成静态网页，然后输入hexo s开启本地服务，最后输入hexo d 上传到github的仓库。

这时打开你的github.io的主页就能看到发布的文章了。
